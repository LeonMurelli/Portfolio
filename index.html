Leon Murelli Silveira Foco, minuciosidade e organização. Expertise em comportamento humano adentrando o mercado de tecnologia.
Basic HTML5 for web markup. Basic CSS3 for style and layout. Basic JavaScript for scripting. Basic React.js for a dynamic interface and JSX integration. Basic Node.js for backend development.

Nascido em 08/12/1986, casado, possuo dois filhos, CNH categoria B, transporte próprio e disponibilidade para viajar. Tenho Experiencia em rotina de administração de microempresa do ramo de saúde, faturamento médico e negociação de contratos adquirida em sua atuação como sócio de clínica de psicologia. Busco um desafio neste nível para continuar o aprendizado destas ferramentas, dominar novas ferramentas utilizadas pelo mercado, integrar-me a uma equipe de desenvolvimento de projetos mais complexos e contribuir para o ramo da tecnologia.
